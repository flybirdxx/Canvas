# Roadmap

## Current Priority

1. Stabilize OmniScript video analysis provider support.
2. Improve upload and upstream video/file reference handling.
3. Add browser-level UI coverage for creating an OmniScript node and rendering analysis results.
4. Harden export and persistence around large media files.

## Removed From Roadmap

Storyboard mode, script nodes, scene nodes, automatic scene splitting, and storyboard MP4 export are no longer planned in this product direction.
